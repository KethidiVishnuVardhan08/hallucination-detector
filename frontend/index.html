How it works

  1. Sample the same prompt N times at temperature > 0
  2. Embed all samples and compute pairwise cosine similarity
  3. High divergence → model is confabulating
  4. Low divergence → model likely knows the answer

Verdicts

Likely grounded < 0.30
Uncertain 0.30 – 0.60
Likely hallucinated > 0.60

Caveats

Self-consistency catches confabulation, not confident falsehoods. If the model is consistently wrong, divergence stays low. N samples = N× API cost.

Based on SelfCheckGPT (Manakul et al., 2023) · No ground truth required at inference time
